# CareerCrawl AI

## Team Name
TechMen

## Project Description
CareerCrawl AI is a web-based platform that crawls technology company websites to collect job postings in the Dallas–Fort Worth (DFW) region. The platform analyzes job descriptions using NLP techniques to extract skills, tools, and experience requirements. It also supports interview preparation and resume skill gap analysis.
